La clase padre es Boton, las clases hijas son boton piso y boton asensor o sea que esas se extends de clase boton
NOSOTRO VAMOS A PODER REALIZAR ESTO 👍
La clase Ascensor es el corazón de EL sistema y es donde se concentrará la mayor parte de la lógica de control.
En esencia, la clase Ascensor se encarga de:
Mantener el estado: Conocer en qué piso está, si está en movimiento, si las puertas están abiertas o cerradas, etc.
Procesar solicitudes: Recibir las solicitudes de los botones (tanto de piso como de dentro del ascensor) y decidir qué acción tomar.
Controlar el movimiento: Mover la cabina del ascensor hacia arriba o hacia abajo.
Gestionar las puertas: Abrir y cerrar las puertas en los pisos correspondientes.
Interactuar con otros componentes: Como por ejemplo, un sistema de alarma, un sistema de iluminación o un sistema de registro de eventos.
